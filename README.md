# 淘宝天猫装修优化——油猴脚本

主要针对淘内后台优化，优化链接http缺失、一键居中、模块去除10像素间距等；店铺装修强制跳转至pc装修；装修后台增加新窗口打开，欢迎交流使用;

## 使用前
在店铺中需在店铺css模块中添加：
请勿重复添加，非前端人员忽略！
``` css

.left465{ margin-left:-465px; width:1920px;}
.up10{ margin-top: -10px;}
.h_jg{ width: 100%; clear: both; display: block;}

```
